This is the submission of Magnifying Moving Objects Project. The members of this group are Aadi Sharma, Jasmeher Singh Chaddha and Satwik Narang. All three have worked on the code at the same time, Jasmeher created the project presentation and Aadi wrote the project report.
This .zip file includes the following:-
1) CV_TERM_PROJECT -> code file written on google colab
2) CV.pptx -> Project presentation
3) CV_Project_Report -> Project Report
4) face(1).mp4 & baby2.mp4 -> input files
5) magnified_face(2).mp4 & magnified_baby2.mp4 -> final output
6) face output.png & baby2 output.png -> variations in input and output at different intervals
